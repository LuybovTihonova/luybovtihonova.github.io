Это нужно сделать один раз!!!
1. Создаем репозиторий на github.com
2. Копируем ссылку созданного репозитория.
3. Клонируем репозиторий к себе на компьютер: git clone ссылка_на_репозиторий


А вот это нужно делать постоянно!!!
1. Рабтает с файломи (правим, добавляем, удаляем, что угодно делаем)
2. Сохраняем все зменения CTRLS + S
2. Фиксируем файлы в Гите: git add .
3. Делаем каммит: git commit -m "Здесь нужно написать любое описание сделанного камита"
4. Отправляем комит на удаленный сервер: git push

5. Можно посмотерть состояние гита: git status
