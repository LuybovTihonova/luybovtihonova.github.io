1. Задача курсовый работы состояла в том, чтобы произвести верстку одностраничного сайта, в соответствии с предоставленным макетом,
   с применением языков программирования HTML5, CSS, JavaScript и загрузке на хостинг GitHub. 
2. Для верстки сайта мною были использованы следующие информационные источники и программные продукты: 
   * Figma - графический редактор для проектирования сайтов, приложений и других дизайнерских продуктов.
   * Visual Studio Code - редактор исходного кода,разработанный Microsoft для Windows, Linux и macOS.
   * GitHub - онлайн-сервис для хранения и синхронизации кода программистов и разработчиков приложений. 
    Главной целью этого сервиса является поддержка совместной разработки проектов и контроля версий.
   * favicon.by. - интернет сервис для создания фавиконок сайтов.
   * Google Fonts — сайт для поиска шрифтов.
   * HTML Academy - интерактивные онлайн-курсы
   * БЭМ методология
